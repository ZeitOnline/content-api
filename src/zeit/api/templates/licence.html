ZEIT ONLINE stellt die API (Application Programming Interface) als öffentliche BETA-Version zur Verfügung. Die API umfasst die Inhalte des ZEIT Archivs seit 1946 sowie Inhalte von ZEIT ONLINE in maschinenlesbarer Form. Mit der API sollen nicht-kommerzielle Erforschung, Analyse und Visualisierung der Inhalte erleichtert werden. Umsetzungen auf Basis der API möchten wir sehr gerne auf diesen Seiten vorstellen, bitte informieren Sie uns unter api@zeit.de.

Die Speicherung und Ausgabe des Volltexts von Artikeln ist zum derzeitigen Stand nicht möglich. Bitte beachten Sie, dass die Beiträge unserer Autoren dem Schutz des Urheberrechts unterliegen. Falls Sie ein Projekt auf Grundlage von Volltextübernahmen planen, möchten wir Sie bitten, mit uns Kontakt aufzunehmen. Es gelten die Allgemeinen Nutzungsbedingungen von ZEIT ONLINE. Für Rückfragen stehen wir gern zur Verfügung.

Wir wünschen einen fruchtbaren Dialog mit allen an der API Interessierten.
Für Vorschläge, technische Fragen oder Anregungen erreichen Sie uns unter api@zeit.de oder @zeitonline_dev, bei Fragen zur kommerziellen Verwendung und Lizenzierung an online-syndication@zeit.de.
